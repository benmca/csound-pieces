*midi-default-connections* [Variable]

A sequence of interface designators to be used to establish associations between logical MIDI connections and the designated interface(s). An interface designator is generally a sequence of length two of interface name strings designating an input and output interface to be associated with the given logical input or output connection, respectively. As a shortcut, a single interface name string may be specified to indicate a bidirectional connection. Logical connection numbers are zero-based and represented indirectly by the position of an interface designator within the overall sequence. NIL may be specified at any point if a particular connection should not be made.

Examples:

;; associate connection 0 bidirectionally with interface "Foo"
("Foo")

;; also associate connection 2 bidirectionally with interface "Bar"
("Foo" NIL "Bar")

;; get input on connection 0 from "Foo" but send output on connection 0 
;; to "Bar" and output on connection 2 to "Baz"
(#("Foo" "Bar") (NIL "Baz"))
