expunge {objects} [Command]

Erases any {objects} marked deleted and removes them from their containers. An expunged object is not recoverable. If {objects} is an undeleted container, its subobjects are searched for deleted objects to expunge. expunge has no effect on an undeleted object itself.

See Also:

delete, hide, mute, undelete, unhide, unmute,
