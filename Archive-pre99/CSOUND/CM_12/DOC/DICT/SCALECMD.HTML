scale {objects} {slot expr}+ [Command]

Maps over {objects} and increments each slot in {slot expr}+ by the value of its corresponding expr. scale is only meaningful for slots containing numeric data.

Example:

Scale the rhythms of all the objects in Foo by an envelope using % positions: 
Stella [Top-Level]: scale foo rhythm (interpl % 0 1 100 2)

See Also:

increment, invert, map, scale, set, transpose unset
