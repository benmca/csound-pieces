note reference &optional scale [Function]

Returns the symbolic note name of reference in scale. Reference may be a note, degree, or pitch. If scale is omitted it defaults to *standard-scale*.

Example:

? (note 'a4)
A4

? (note 440.0)
A4

? (note 69)
A4
