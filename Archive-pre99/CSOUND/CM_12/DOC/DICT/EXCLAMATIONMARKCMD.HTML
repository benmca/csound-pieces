! {number | ?}+ [Command]

Re-executes previous command or lisp input. If {event} is not specified it defaults to the last input. If {event} is ? the history list is printed. If {event} is a negative number it references events backward from the current event. If {event} is positive it references that event within a window of 50 previous events.

Example:

? (stella)
Stella [Top-Level]: (setf x (random 10000))
5642
Stella [Top-Level]: !

Stella [Top-Level]: ,x
2050
Stella [Top-Level]: ! ? 
[1]     (SETF X (RANDOM 10000))
[2]     (!-CMD "")
[3]     X
Stella [Top-Level]:
