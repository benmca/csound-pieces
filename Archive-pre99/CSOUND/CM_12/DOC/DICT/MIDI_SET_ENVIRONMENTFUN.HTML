midi-set-environment chanmap routemap connections [Function]

Set the current MIDI environment according to chanmap, routemap, and connections. This is almost equivalent to calling midi-set-channel-map, midi-set-connections, and midi-set-route-map but also performs a consistency check.

See Also:

midi-set-channel-map, midi-set-connections, midi-set-route-map
