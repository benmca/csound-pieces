midifile-play &optional pathname &key :start :end :timescale :headstart :port [Function]
Plays the contents of the MIDI file pathname. If pathname is not specified, it defaults to the value of *midifile-default-pathname*. :start is the start time in the file to begin listening at. :end is the time in the file to stop listening at. :timescale is a tempo scaler to apply to the messages and defaults to 1.0. :headstart is the number of milliseconds to delay the onset of sound by, and defaults to 1000 (1.0 seconds). :port is the MIDI port to open if one is not currently open.

See Also:

*midifile-default-pathname*, *midifile-player*, *midifile-player-args*
