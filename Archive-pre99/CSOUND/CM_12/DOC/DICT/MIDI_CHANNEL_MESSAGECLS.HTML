midi-channel-message [Class]

Represents a MIDI channel message of any type. Slots declared by midi-channel-message:

type {integer}
The MIDI message type. Valid types are +midi-note-off+, +midi-note-on+, +midi-key-pressure+, +midi-control-change+, +midi-program-change+, +midi-channel-pressure+, +midi-pitch-bend+.
channel {integer}
The MIDI channel of the message. Defaults to 0.
data1 {integer}
First data byte of the message.
data2 {integer}
An optional second data byte of the message.

See Also:

midi-note, midi-message
