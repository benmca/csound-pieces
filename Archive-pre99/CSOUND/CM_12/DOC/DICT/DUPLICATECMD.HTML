duplicate {container} {newname}* [Command]

Copies {container} to an optional {newname} and places the new object at the end of the top-level container.

Example:

Duplicate Foo to new object Foo-Copy: 
Stella [Top-Level]: duplicate foo
Duplicate foo to new object Bar: 
Stella [Top-Level]: duplicate foo bar
