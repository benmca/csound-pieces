import {file} {option value}* [Command]

Imports the contents of {file} into Stella. Currently supports MIDI, MusicKit and CSound files. The following {option value} pairs apply to importing MIDI files:

track {number}
Specifies the track (zero-based) in a level 1 midifile to be imported. Defaults to 0.
container {object}
Specifies the container to receive imported MIDI data. Defaults to a new thread.
meta {t | nil}
Determines if MIDI meta messages will be imported. Defaults to nil.
note-type {note | pitch | degree}
Determines how MIDI key numbers are to be interpreted. Defaults to note.
note-off-stack {t | nil}
Determines how multiple note on/off message with identical channels and key numbers are paired. The default value is nil, which means that the current note off is matched with the earliest pending note on with the same channel and key number. A value of t means to match the current note off with the most recent pending note on with the same key number and channel.
