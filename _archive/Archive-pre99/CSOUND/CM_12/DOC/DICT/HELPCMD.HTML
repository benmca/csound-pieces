help {topic}+ {option value}* [Command]

Shows help documentation matching {topic}. If {topic} matches more than one topic the list of topics is displayed.

help supports the following {option value} pairs:

exact {t | nil}
If exact is nil (the default), all topics containing {topic} are listed, otherwise only help exactly matching {topic} is displayed.
browser {t | nil}
If t (the default) then help uses an external HTML browser to show help, otherwise an ascii version of the help text is printed to the terminal. Help currently supports Netscape on Mac and SGI, and OmniWeb on NextStep.
domain {CM | CLM}
Sets documentation domain to CM or CLM. The domain is sticky and defaults to CM.
