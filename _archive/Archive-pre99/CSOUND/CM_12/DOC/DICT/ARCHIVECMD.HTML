archive {objects} {file} {option value}* [Command]

Saves {objects} and all subobjects to {file}. The default file extension for the archive is .cm. Objects marked deleted are skipped during archive processing. Care should be taken when archiving an object in several situations:

  • Archiving a container without its superiors will cause these links to be broken when the archive file is reloaded.
  • Archiving an algorithm may be impossible or undesirable if the algorithm contains free variables, or it was created in a loop, or it depends on a lexical closure. Algorithms are most easily developed by using a text editor; that way the environment that the algorithm depends on can be included with the algorithm in the same lisp file.
