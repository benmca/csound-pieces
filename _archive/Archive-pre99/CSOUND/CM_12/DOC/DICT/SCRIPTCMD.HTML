script {file} {option value}* [Command]

Processes {file} as if its contents were typed directly to Stella. {file} may contain both Lisp forms and commands expressions. Script files have a default extension of ".tl" (tl is the name of stella's command dispatching loop) to distinguish them from normal .lisp files, which cannot contain command expressions. Script files are useful for storing complicated editing sequences, or command sequences that are issued repeatedly. The following {OPTION value} pairs apply to script:

commands {list}
Commands to use, default Stella.
output {stream}
Output stream, default terminal
echo {boolean}
If T (default) echo input.
prompt {string}
Show lines prefaced by {string}
