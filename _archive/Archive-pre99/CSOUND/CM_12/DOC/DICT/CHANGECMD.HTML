change {objects} {newclass} {new old}* [Command]

Changes the class of {objects} to {newclass}. Slots that exist in both the old and new object keep the same values. Slots in the new class that do not exist in the old class are initialized according to the optional {new old}* list, where each new slot in the redefined instance receives the corresponding old slot value from the old instance.

Example:

Change all objects in Foo to be Pluck objects and set the new freq slot to the old note value: 
Stella [Top-Level]: change * pluck freq note
Change the class of only midi-notes: 
Stella [Top-Level]: {map * when (eq $class midi-note)
       change pluck freq note}
