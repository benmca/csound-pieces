clm-note [Class]

An event class that automatically produces CLM sound file and score file events for all subclasses. Subclasses are automatically declared using the definstrument macro.

clm-note declares the following slots:

pars {list}
Holds a list of parameter slot descriptions for passing slot values to the CLM instrument or score file. Pars is internally managed the by definstrument macro.
Additional slots are inherited from note.
