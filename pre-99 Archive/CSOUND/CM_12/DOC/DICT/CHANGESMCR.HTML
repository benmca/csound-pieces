changes {start value}* {end value}* {step value}* [Macro]

Returns one or more values suitable for the rotation pattern's change option. start, end, step, and width may be numbers or lists of numbers. These values are "spread" across as many changes as necessary to represent the longest list of values.

Example:

? (setf x (items a b c d in rotation
                 change (changes start '(0 1) width 2)))
#<ROTATIONAL-ITEM-STREAM 123424>

? (loop repeat 8 collect (read-items x))
((A B C D) (C D A B) (C B A D) (A D C B) (A B C D)
 (C D A B) (C B A D) (A D C B))
