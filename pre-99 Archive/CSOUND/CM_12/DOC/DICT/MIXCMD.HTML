mix {containers} {offsets} {stream}+ {option value}* [Command]

Sends the mixed output from {containers} to {stream} or to the currently open stream if {stream} is not specified. If {offsets} is a single number it becomes the start time for all {containers}. If {offsets} is a series of numbers seperated by comma they are applied to each corresponding object in {containers}. Otherwise, if {offsets} is *, start times for each {container} are individially prompted for. Start time offsets specified to mix affect only the current listening pass.

See open for a list of available {option value} pairs.

Example:

Mix Foo and Bar offset to 1.5 seconds, don't play the output stream: 
Stella [Top-Level]: mix foo,bar 1.5 play nil
Mix Foo and Bar at times .1 and 2 to test.midi, play it too. 
Stella [Top-Level]: mix foo,bar .1,2 test.midi play t
Mix Foo and Bar with prompted offsets: 
Stella [Top-Level]: mix foo,bar
Start time offset: (<cr>=None) *
Foo start time offset: (<cr>=None) 
Bar start time offset: (<cr>=None) 5
