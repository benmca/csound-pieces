find-object name &optional errorp [Function]
#!name [Read Macro]

Returns the object named name. If errorp is t an error is signaled if the object does not exist.#! can be used inside the definition of a thread or merge to the referenced object a subobject in the new definition.

Example:

? (thread foo ())
#<THREAD Foo>

? #!foo
#<THREAD Foo>

? (thread bar () (object rest) #!foo (object midi-note) #!foo)
#<THREAD Bar>

? (list-object #!bar)
Bar:
     1. #<REST unset 132200061> 
     2. #<THREAD: Foo> 
     3. #<MIDI-NOTE |---|------|------| 64| 0|> 
     4. #<THREAD: Foo>
