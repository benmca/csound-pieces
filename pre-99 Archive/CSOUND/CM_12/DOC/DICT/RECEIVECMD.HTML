receive {function} {option value}* [Command]

A general purpose midi input utility. {function} is either the name of a function that accepts two values, the message and its millisecond time, or one of three receiving modes:

through
sends all received messages directly to the midi port(s)
record
captures midi data to a thread for later editing.
print
prints the messages and times. If a function is specified it is passed two arguments, the MIDI message and its millisecond time.
receive supports the following {option value} pairs:

keynum {keynum}
Stop receving if {keynum} is pressed.
duration {time}
Stop receive after {time} seconds.
time {t | nil | time}
If time is t (the default) then MIDI time is offset from the first message received, otherwise if time is a number the clock is reset to that value, otherwise the clock is not reset. Calling receive without arguments stops asynchronous receiving in MCL and ACL Common Lisp. Asynchronous receiving is not supported in CLISP or AKCL.
