delete {objects} {option value}* [Command]

Marks {objects} as deleted. A deleted object It still occupies a position in its container and appears in listings marked "deleted", but does not appear in event processing and is not saved by the archive command.

delete supports the following {option value} pairs:

recurse {t | nil}
Recursive delete. Default is nil.
expunge {t | nil}
Destroy deleted objects. Default is nil.
