cl {file} {option value}* [Command]

Conditionally compiles and loads the binary version of {file}. A binary is compiled if:

  • there is currently no binary version.
  • the binary version is out of date.
  • force-recompile is t

If {file} does not include a filename extension cl searches for a version of {file} using the following order of extensions: .cm, .ins, .lisp

cl supports the following {option value} pairs

force-recompile {t | nil}
Compile even if binary is uptodate.
output-file {pathname}
Specify non-default binary location.
verbose {t | nil}
Print compilation/load progress.

Example:

Compile and load test.lisp from the current working directory: 
Stella [Top-Level]: cl test
